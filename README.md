lila-openingexplorer3
=====================

Goal: Implement personal opening explorer. Likely to become a rewrite of the
existing opening explorer.

Explorer
--------

## Hashed

* Position (incl. variant)
* Player
* White/Black

## Lexicographic

* Date
* Move

## Multiple select

* Speed
* Rated/Casual

## Value

* Win/Draw/Loss
* Rating
* Games: IDs, Opponents with ratings

Player
------

* Last game ID
* Last pulled
* Name
* Title
* Erased

Game
----

* White player
* White rating
* Black player
* Black rating
